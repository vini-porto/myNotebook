# What is SQL
Is a programming language witch purpose is processing information in a [[Relational Database]]. It is declarative, witch means that you use more descriptive commands. 

---
# SQL is Declarative
[[Declarative Programming]]: You describe in a more direct way what you want.

**e.g.**
- [[Procedural Programming]]: "Walk to the kichen, open the fridge, get milk".
- [[Declarative Programming]]: "Bring me milk"
